# LedgerFlow

LedgerFlow is a web-based accounting tool designed to streamline the workflow for accountants by bridging the gap between bank statements and Tally.

## 🚀 Features
* **Easy Upload:** Quickly parse standard bank statements.
* **Instant Conversion:** Automatically convert statement data into Tally-compatible XML files.
* **Seamless Import:** Directly import the generated XML into Tally ERP or Tally Prime without manual data entry.

## 💻 How to Use
1. Open the website in your browser.
2. Upload or paste your bank statement file.
3. Click convert to process the transactions.
4. Download the generated `.xml` file and import it straight into Tally.
